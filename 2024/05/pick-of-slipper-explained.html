Pick of the Slipper Explained

What is Pick of the Slipper?

Pick of the Slipper is an exciting multi-day pick game event where players aim to find Dorothy's Ruby Slippers hidden beneath one of the symbols on the game board. By uncovering the Ruby Slippers in each stage, players can win fantastic prizes. After conquering the final stage, daring players can replay the stages for even bigger rewards.

Upon entering the Pick Game screen, a board of symbols awaits. For each Pick (represented by Stars) a player has, they can select one symbol to reveal what lies beneath. If the Ruby Slippers are found, the stage is completed, a prize is awarded, and the player advances to the next stage with a grander prize awaiting.

Players need not fret if they leave the Pick Game before locating the Ruby Slippers, as their progress is saved for their return.

Accessing Pick of the Slipper

To access the Pick Game at any time during the event, players simply need to tap the star-shaped button on the bottom menu, reminiscent of Glynda's wand.

Making Picks

To make a Pick in the Pick of the Slipper game, players select one of the many symbols on the game board. Picks are represented by stars, and each star allows one symbol selection. The number of available Picks is displayed in the upper right-hand corner of the Pick Game screen.

Earning Picks

Picks are earned by spinning on any game machine! Every few spins, star symbols will drop from the reels and fill the meter on the left side of the screen. Each time the meter is filled, the player earns one Pick. Higher bets cause each symbol to fill the meter more quickly! However, players can only hold a limited number of Picks at a time, so using them is encouraged.

Prizes

At the end of every stage, a credit prize awaits the player. Furthermore, other rewards can be uncovered behind some of the symbols in each stage, including:

  • Instant Credit prizes
  • Reward Up prizes to increase the current stage's Ruby Slipper reward
  • Lucky Card Packs
  • Extra Picks for more turns!

Event Duration

The duration of each Pick of the Slipper event varies, but players can check the timer above the game board in the Pick Game screen to see how much time remains in the current event.
