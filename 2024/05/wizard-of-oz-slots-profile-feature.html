Wizard of Oz Slots Profile Feature

The Profile feature in Zynga's slot games allows players to customize their personal information, track gameplay statistics, and view other players' profiles. Here's an overview of this feature:

What is the Profile Feature?

The Profile feature enables players to edit their own information, including profile photo, name, status, location, and gender. It also displays gameplay stats such as current level, VIP tier, favorite machine, and biggest win. Players can view their stats across multiple Zynga slot games if they are connected to the Loyalty Lounge program using the same email address.

Accessing the Profile Feature

To access the Profile feature, players can tap the hamburger menu button in the upper-right corner of the game page and then tap the "My Profile" button.

Profile Tab

The Profile tab consists of two sections:

  1. Manage Your Own Profile: Here, players can edit their profile photo (using their Facebook profile photo or selecting from an avatar library), set a mood status, update their name/nickname, location, and gender.
  2. Viewable Section: This section displays the player's current VIP tier, the year they joined the Loyalty Lounge, their Loyalty Lounge ID, current game level, favorite machine (based on spins in the last 30 days), and biggest win (in coins) in the past 30 days.

Games Tab

The Games tab displays gameplay statistics across multiple Zynga slot games, if the player is connected to the Loyalty Lounge using the same email address. It shows the biggest win (in coins) from all linked accounts within the past 30 days, the highest game level achieved in each game, the favorite machine (based on spins in the last 30 days), and the date each game was last played.

Viewing Other Players' Profiles

Players can view other players' profiles by clicking on their photo in the "Send Gift" section of the Yellow Brick Road and selecting "View Profile." If a player's profile contains abuse or profanity, it can be reported using the "REPORT" button on the lower-right side of their profile page.

Loyalty Lounge Connection

To customize the profile photo and information, players need to sign up for the Loyalty Lounge program. Any changes made to the profile in one game will be synced across all Zynga slot games connected to the Loyalty Lounge using the same email address.

By providing a centralized profile feature, Zynga allows players to personalize their gaming experience, track their progress, and connect with other players within the same ecosystem.
